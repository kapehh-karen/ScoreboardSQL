ScoreboardSQL
=============

Использует PluginManager, Vault

Конфиг:
<pre>
connect:
  ip: <host> # хост с БД
  db: <name> # название базы
  login: <login> # логин юзера
  password: <passwd> # его пароль
timer:
  ticks: <ticks> # количество тиков для обновления инфы (1000 это 1 минута, примерно)
</pre>
